# CVEstigator
A web tool for multiple CVEs description and score lookup

Just paste text containing CVE identifiers into the text field and click "Lookup CVEs" or press Ctrl+Enter to get structured vulnerability data from CVE.org!
The tool provides a deduplicated list with CVE number, description, scores in different CVSS versions and links to CVE.org and NIST.

**Remember:** 
This is only a tool and it may be wrong.

CVEs can have multiple scores, so please check the provided links to CVE details in different sites to find more info.
